== Description ==

Nous avons associé mobile money, woocommerce et marketing d’affiliation afin de permettre aux acteurs du e-commerce africains de briser ces barrières de moyen de paiement sur les plateformes multi-vendeurs.

== Installation ==

1. Upload the entire `SAMM: Solution Africaine pour Marchés Multi-vendeurs` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find **Samm** menu in your WordPress admin screen.
